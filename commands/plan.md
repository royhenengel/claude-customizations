Create requirements and executable plans for features.

## What This Does

1. Checks prerequisites (planning/ exists)
2. Offers brainstorm for unclear requirements (optional)
3. Creates feature directory in `planning/specs/`
4. Generates spec.md, research.md, and PLAN.md
5. Updates STATE.md to planning stage

## Usage

```bash
/plan
/plan [feature-name]
```

## Invocation

Load the my-workflow skill and execute the plan workflow:

```text
@skills/my-workflow/SKILL.md
@skills/my-workflow/workflows/plan.md

Execute the /plan workflow as documented.
```
