# Formatting Rules

**Domain:** Output formatting - list spacing, visual structure, and presentation standards.

Rules governing how Claude formats output in this repository.

## Lists

CRITICAL: When presenting lists (numbered or bulleted) where each item has substantive content (more than a few words), add a blank line between items.

**Correct:**

1. First item with explanation

2. Second item with explanation

3. Third item with explanation

**Incorrect:**

1. First item with explanation
2. Second item with explanation
3. Third item with explanation

Exception: Short reference lists (single words, file names, brief labels) may use compact format without blank lines.
